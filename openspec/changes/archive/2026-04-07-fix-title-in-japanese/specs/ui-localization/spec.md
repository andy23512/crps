## MODIFIED Requirements

### Requirement: UI supports Japanese language mode

The system SHALL provide a Japanese language mode for player-facing UI text in addition to English, including the localized application title.

#### Scenario: Player selects Japanese language

- **WHEN** the player changes language to Japanese
- **THEN** the UI SHALL render localized Japanese text for supported labels, controls, status messages, and the application title
- **AND** gameplay behavior SHALL remain unchanged
